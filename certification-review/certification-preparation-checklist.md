# Certification Preparation Checklist

This checklist summarizes practical review points for certification preparation in mechanical product development.

## 1. Product Information

- Product name
- Model name
- Product configuration
- Rated voltage and power
- Operating temperature range
- Intended installation environment
- Applicable market or region
- Product label and marking requirements

## 2. Drawing and Document Control

- Check drawing revision
- Check part list and BOM revision
- Check assembly drawing
- Check critical dimensions
- Check material specifications
- Check surface treatment information
- Check document consistency between drawings, BOM, and technical files

## 3. Material Documentation

- Verify material grade
- Check material certificate
- Check flame rating or certification status if required
- Check gasket, O-ring, label, and window material information
- Check adhesive, sealant, and coating documentation
- Confirm supplier and manufacturer information

## 4. Marking Review

- Check product nameplate information
- Check model name
- Check rating information
- Check warning statements
- Check certification marks
- Check serial number or lot number format
- Check label durability and attachment method

## 5. Test Sample Preparation

- Check sample configuration
- Check assembly condition
- Check screw tightening condition
- Check sealing condition
- Check cable entry or connector condition
- Check representative worst-case sample
- Check pre-test inspection records

## 6. Technical File Organization

- Product description
- Drawing package
- BOM
- Material documents
- Test reports
- Risk review notes
- Installation instructions
- User manual
- Marking drawing
- Change history

## 7. Review Points Before Submission

- Confirm all submitted documents use the same model name
- Confirm drawing revisions match actual samples
- Confirm material documents match applied parts
- Confirm label information matches certification application
- Confirm test samples match submitted drawings
- Confirm any design changes are recorded

## Notes

This document is a generalized engineering checklist.  
It does not include confidential drawings, customer information, internal test reports, or proprietary project data.
